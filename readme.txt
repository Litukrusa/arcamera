AR MIRROR — инструкция по запуску сайта локально

Сначала установите редактор Visual Studio Code.
Откройте сайт: https://code.visualstudio.com/
Скачайте установщик для вашей операционной системы.
Установите программу, следуя инструкциям установщика.
------------------------------------------------------------------------
После установки Visual Studio Code установите расширение Live Server.
Откройте Visual Studio Code.
Нажмите на раздел Extensions (Расширения) в левой панели ииспользуйте сочетание клавиш Ctrl + Shift + X.
Найдите расширение Live Server (автор Ritwick Dey).
Нажмите кнопку Install (Установить).
------------------------------------------------------------------------
Распаковать архив с проектом.
Открыть в Visual Studio Code папку проекта.
Нажмите:
File → Open Folder
Выберите папку проекта AR MIRROR.
Нажмите кнопку Открыть.
------------------------------------------------------------------------
Запуск локального сайта
После открытия проекта выполнить следующие действия.
1.  Найдите файл:
index.html
2.  Нажмите на него правой кнопкой мыши.
3.  Выберите пункт:
Open with Live Server
------------------------------------------------------------------------
Открыть сайт в браузере
После запуска Live Server сайт автоматически откроется в браузере.
Обычно адрес будет таким:
http://127.0.0.1:5500/index.html // http://localhost:5500/index.html
------------------------------------------------------------------------
Доступ к камере
Когда сайт откроется, браузер может попросить доступ к камере.
Чтобы AR-примерка работала:
1.  Нажмите кнопку Allow (Разрешить).
2.  После этого камера включится и сайт сможет использовать AR.
